# Trabalho Prático — Testes de Componente em Python

Este projeto contém um subsistema de maker lab já implementado, com testes de unidade prontos.

## Objetivo
O trabalho dos alunos é criar os testes de componente em `tests/components/`, validando a colaboração real entre as classes do subsistema.

## Como instalar
```bash
pip install -r requirements.txt
```

## Como executar os testes de unidade
```bash
pytest tests/unit
```

## Como executar todos os testes
```bash
pytest
```

## Pasta que os alunos devem completar
```text
tests/components/
```
